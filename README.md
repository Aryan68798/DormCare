# 🏨 DormCare

DormCare is a web platform designed to simplify hostel life for students by providing easy access to essential hostel services in one place. Students can book room cleaning, order meals from the hostel mess with room delivery, and request other hostel services through a simple and user-friendly interface.

The goal of this project is to improve convenience, organization, and communication between students and hostel management.

---

## 🚀 Features

- 🧹 **Room Cleaning Booking**  
  Students can schedule cleaning services for their rooms.

- 🛠️ **Maintenance Requests**  
  Report issues such as electrical, plumbing, or other room problems.

- 📋 **Service Dashboard**  
  View all requests and track their status.

- 📱 **Simple and User-Friendly Interface**  
  Designed for quick and easy access to hostel services.
